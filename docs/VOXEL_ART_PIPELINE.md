# Voxel Art Pipeline

- Author gameplay structures as true runtime voxels inside the world grid.
- Author commanders, weapons, and props in a voxel authoring tool such as Blockbench or MagicaVoxel.
- Export textured meshes into `assets/models/` and reuse the shared atlas in `assets/textures/voxels/`.
- Keep material readability stronger than realism; silhouette and breakability matter more than surface detail.
- Model pieces with chunky sub-forms so destruction reads as many smaller fragments instead of one giant block popping away.
